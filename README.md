# Decode Labs Data Science Internship

Welcome to my repository for the **Decode Labs Data Science Internship Program**. This repository hosts all the projects and assignments completed during the internship, demonstrating end-to-end data pipelines, exploratory data analysis (EDA), preprocessing, and machine learning model development.

---

## Repository Projects

The repository is organized into distinct projects:

### 🚀 [Project 01: E-Commerce Data Cleaning & Preprocessing](file:///Users/shivampatidar/Downloads/DecodeLabs-Internship/Project-01/README.md)
* **Goal:** Perform comprehensive cleaning, outlier treatment, feature engineering, and encoding on noisy e-commerce transaction data.
* **Key Tasks:** Missing value imputation, IQR capping for outliers, DateTime extraction, and mixed category encoding (Label & One-Hot Encoding).
* **Deliverables:** Cleaned dataset and fully documented analysis notebook.
* **[View Project 01 README](file:///Users/shivampatidar/Downloads/DecodeLabs-Internship/Project-01/README.md)**

---

### 💳 [Project 02: Credit Card Default Prediction (Supervised Learning)](file:///Users/shivampatidar/Downloads/DecodeLabs-Internship/Project-02/README.md)
* **Goal:** Develop a binary classification pipeline to predict if credit card clients will default on payments next month.
* **Key Tasks:** Exploratory data analysis, leak-free train-test splitting, feature scaling, class balancing using SMOTE, model training, and performance evaluation.
* **Deliverables:** Predictive classification models, serialization files for scalers/models, and evaluation notebook.
* **[View Project 02 README](file:///Users/shivampatidar/Downloads/DecodeLabs-Internship/Project-02/README.md)**

## Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/ShivamPatidar03/DecodeLabs-Internship.git
cd DecodeLabs-Internship
```

### 2. Set Up a Virtual Environment (Optional but Recommended)
```bash
# Create a virtual environment
python -m venv venv

# Activate it (Mac/Linux)
source venv/bin/activate

# Activate it (Windows)
# venv\Scripts\activate
```

### 3. Install Dependencies
Make sure you have the following packages installed:
```bash
pip install pandas numpy scikit-learn imbalanced-learn matplotlib seaborn jupyter
```
